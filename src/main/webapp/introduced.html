软帝药店管理系统-系统介绍

系统功能：

  1. 客户管理：记录每个客户的个人信息
  2. 药品管理：按照药品分类进行药品的库存数量，信息管理
  3. 订单管理：记录药店的每一单数据
  4. 店员管理：管理员可以增删改查店员，进行店员信息的管理

技术选型：

  1. 开发语言：Java/HTML/CSS/JavaScript/Jsp
  2. 前端框架：Layui
  3. 数据库： Mysql
  4. 数据库连接池：Druid。
  5. 工具类：Fastjson,log4j,poi,MyJdbc
  6. 更多……
